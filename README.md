# 智绘旅途

> AI 技术重构个性化旅行体验，千人千面专属智慧旅游管家。

## 项目简介

智绘旅途，致力于通过 AI 技术重构个性化旅行体验，精准解析用户偏好，动态生成涵盖行程规划、景点推荐、食宿安排的全链条方案。聚焦青壮年及家庭客群，解决传统旅游服务标准化与用户需求差异化的矛盾，以智能管家全程陪伴、实时动态调整为核心，让旅行摆脱攻略束缚，实现“灵活出行，智在不设限”的便捷体验，推动旅游服务从“千人一面”向“千人千面”升级。

## 主要板块

1. **用户画像**  
   整合多渠道数据，精准描绘用户偏好。

2. **AI智能生成方案**  
   选定旅行标签，一键生成多样化可视化全链条方案。

3. **旅行私人管家**  
   方案确认后自动提醒并跳转订车次/酒店/门票，AI 智能推荐周边游。

## 快速启动

```bash
npm install
npm start
```

## 部署到 GitHub Pages

1. 修改 `package.json` 的 `homepage` 字段为你的 Pages 路径，例如 `"homepage": "https://your-username.github.io/your-repo-name"`
2. 执行
   ```bash
   npm run build
   ```
3. 将 `build` 目录上传到 GitHub Pages 或使用 gh-pages 工具自动部署

## Logo/图片及广告位

如无现成图片，请将 logo、景点图片、广告图片分别置于 `public/logo.png`、`public/ad.png` 等路径下或在主界面适当添加。
